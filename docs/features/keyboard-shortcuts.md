# Keyboard Shortcuts

FlowMail is designed to be keyboard-first. You can navigate the entire app, process your inbox, and manage tasks without ever reaching for the mouse.

## Global Shortcuts

These work from anywhere in the app:

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | Open the command palette |
| `Escape` | Close the current panel, modal, or search |
| `Cmd+Z` | Undo last trash action |
| `1` – `8` | Jump to a view (see below) |

## View Navigation

Press a number key to jump directly to any view:

| Key | View |
|-----|------|
| `1` | Triage |
| `2` | Focus Queue |
| `3` | Cards |
| `4` | Memory |
| `5` | Learning |
| `6` | Settings |
| `7` | Archive |
| `8` | Trash |

## Triage Shortcuts

When processing your inbox:

| Shortcut | Action |
|----------|--------|
| `j` | Move to next email |
| `k` | Move to previous email |
| `e` | Archive selected email |
| `r` | Reply to selected email |
| `c` | Open card picker for selected email |
| `s` | Sync emails |
| `/` | Open search |
| `Shift+Arrow` | Multi-select emails |
| `Cmd+Click` | Toggle individual email selection |

## Cards Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | Create a new card |

## Command Palette

Press `Cmd+K` to open the command palette — a quick-access search bar that lets you jump to any view, card, or action without leaving the keyboard. Start typing to filter results, use arrow keys to navigate, and press Enter to select.

## Tips

- **Stay in flow**: Use `j`/`k` to move through your inbox and `e` to archive — you can clear your triage queue without touching the mouse.
- **Quick context switching**: Number keys let you bounce between views instantly. Press `1` to go back to triage, `2` to check your tasks, `3` to see your cards.
- **Batch processing**: Hold `Shift` and use arrow keys to select multiple emails, then apply actions to all of them at once.
- **Command palette for everything**: If you forget a shortcut or want to jump to a specific card, `Cmd+K` is your universal search.
- **Shortcuts are also listed in Settings**: Open **Settings > Preferences** to see the full shortcut reference within the app.
